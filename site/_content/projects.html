Projects

This is a curated list of some of my projects. You can find a more exahustive list if you scroll through my GitHub.

  • yar: an incremental static site generator written in Rust. It's relatively fast, dead simple to use, and powers this website.
  • pint: a user friendly parser combinator library for Python. Parsers are fun, higher order functions are cool, so it stands to reason that combining them is probably even better.
  • Lite: a friendly imperative toy programming language running on a bytecode VM.
  • gol: Conway's Game of Life implemented in Rust, capable of running on the web via WASM.
  • dragonfly-mainframe (contributor): A Python FastAPI distributed computing control server that powers Vipyr Security's security infrastructure.

Additionally, while it's less of a tangible project, I do self-host and administrate a handful of services for my personal use, including

  • Immich
  • Forgejo
  • Mealie
  • Komodo
